# activity-recognition

This repository exists to make useful activity and contribution signals in the
Plone ecosystem easier to find, review, and reuse.

Instead of leaving public GitHub data scattered across many repositories,
searches, and workflow logs, we collect selected views regularly and publish
them as stable static outputs. That makes the information easier to browse in a
browser, easier to reference in conversations and planning, and easier to
consume from external services.

See the published and source documentation in [`docs/`](./docs/).
